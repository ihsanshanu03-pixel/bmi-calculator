# 🧮 BMI Calculator

A simple and responsive **Body Mass Index (BMI) Calculator** built using **HTML, CSS, and JavaScript**.

This project calculates BMI based on height (cm) and weight (kg) and displays the health category according to standard BMI ranges.

---

## 🚀 Features

- Clean two-card layout UI
- BMI calculation using Metric system (cm & kg)
- Displays:
  - Underweight
  - Healthy
  - Overweight
  - Obesity
- Styled using modern CSS
- Responsive card-based design
- Instant result display with color indication

## 🧠 How BMI is Calculated

BMI Formula:

BMI = weight (kg) / (height (m) × height (m))


### BMI Categories

| Category      | BMI Range        |
|--------------|------------------|
| Underweight  | Below 18.5       |
| Healthy      | 18.5 – 24.9      |
| Overweight   | 25.0 – 29.9      |
| Obesity      | 30.0 or above    |
